# 🏠 Aura Space Interiors Website

A premium **Interior Design Portfolio Website** built using HTML, CSS, and JavaScript. This project showcases a modern, elegant UI with smooth animations, theme customization, and interactive components tailored for an interior design business.



## 🚀 Live Preview

👉 (https://future-fs-03-green.vercel.app/)



## 📌 Project Overview

**Aura Space Interiors** is a Bangalore-based interior design studio website designed to:

* Showcase services and projects
* Provide a premium user experience
* Allow users to explore designs visually
* Enable easy contact and consultation booking

The website follows a **modern luxury design system** with smooth transitions and rich UI interactions.



## ✨ Features

### 🎨 UI & Design

* Multi-theme support (Dark, Light, Forest, Slate, Rose, Sand) 
* Elegant typography using Google Fonts 
* Fully responsive design (mobile + desktop)

### 🧭 Navigation

* Sticky navbar with scroll effect 
* Mobile hamburger menu
* Smooth scrolling navigation

### 🖼️ Hero Section

* Auto slideshow with indicators 
* Parallax scrolling effect
* Animated statistics counter

### 📂 Projects Section

* Dynamic project filtering (Living, Kitchen, Bedroom, etc.) 
* Interactive hover effects

### 🎯 User Experience

* Scroll reveal animations
* Testimonials slider
* Back-to-top button
* Newsletter interaction

### 📩 Contact System

* Interactive contact form with success feedback 


## 🛠️ Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Styling:** Custom CSS with variables & themes 
* **Fonts:** Google Fonts (Cormorant, DM Sans, Playfair Display) 
* **Animations:** CSS transitions + JavaScript
